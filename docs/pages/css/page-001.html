Усечение текста до определенного количества строк

Усечь текст до длины одной строки можно следующим образом:


.truncate {
  text-overflow: ellipsis;
  overflow: hidden;
  white-space: nowrap;
}

Однако, данный способ не подойдет, если необходимо обрезать текст например после второй или третьей строки.

Решить задачу можно с помощью свойства line-clamp в такой комбинации:


.line-clamp {
  display: -webkit-box;
  -webkit-line-clamp: 3;
  -webkit-box-orient: vertical;  
  overflow: hidden;
}

где -webkit-line-clamp: 3; - количество строк, после которых текст будет обрезан.

Свойство не поддерживается в IE и некоторых мобильных браузерах.
